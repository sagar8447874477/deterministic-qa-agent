# Deterministic CSV Analytics QA Agent

A fully local CSV analytics assistant built using:

- Streamlit
- Pandas
- Deterministic NLP
- Python

## Features

- Upload CSV
- Ask natural language questions
- Dataset summaries
- Aggregations
- Groupby analytics
- Fully offline
- No LLM APIs

## Example Questions

- how many rows
- show columns
- average sales
- total revenue
- sales by region
